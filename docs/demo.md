# 从作业压缩包到 Excel：完整示例

这一页用虚构数据演示一次典型的“压缩包批阅模式”。你可以先看这里，判断这个工具是否适合自己的课程。

这个流程最适合图片、视频、PDF、PPT、报告、源文件或压缩包混合提交的作业，尤其是艺术、设计、影视、动画、数字媒体、三维、视觉传达等课程。纯文字作业也可以使用，但通常不会默认走作品总览图；工具会为每个学生准备文本评阅材料，并可合并成一个全班文字评阅包，方便先整体阅读和分档。

传统方式往往需要反复解压、逐个打开文件、手动整理表格。本工具会先把学生提交整理成本地评阅材料，再生成作品总览图、评阅记录和 Excel。

## 示例输入

真实使用时，你通常会先从泛雅 / 超星下载全班作业压缩包，然后放到：

```text
tmp/bundle/
```

压缩包里推荐是“每个学生一个文件夹”的结构：

```text
example-course-assignment.zip
├─ 20240001_示例学生A/
│  ├─ final-video.mp4
│  ├─ layout-board.jpg
│  └─ report.pdf
├─ 20240002_示例学生B/
│  ├─ poster.png
│  ├─ process.pptx
│  └─ source.zip
└─ local-001_示例学生C/
   ├─ render-01.jpg
   └─ reflection.docx
```

如果学生文件夹里还有二级压缩包，工具也会尽量继续解压和识别。无法自动读取的文件会被标记为需要人工查看，不会强行评分。

## 评阅过程

1. 在 Codex 或其他本地 AI 助手中输入：

   ```text
   开始泛雅评阅
   ```

2. 选择“压缩包批阅模式”。

3. 确认课程、作业、评分标准和评阅记录文件。

   评分标准会根据作业要求生成或复用。用户确认前，工具不会开始评阅学生。

4. 工具解压作业压缩包，并生成真实学生名单。

   学生名单来自压缩包或网页列表，不应使用手写的示例名单进入正式评阅。

5. 用户确认是否同步网页上已经完成的学生状态。

   如果只是本地评阅，也可以选择不同步网页状态。

6. 用户确认是否有不评阅的学生。

   被跳过的学生会单独记录，不会混入已评阅学生。

7. 工具准备可查看材料。

   常见处理包括：

   - 视频：抽取代表帧；
   - PDF：优先提取可读文字；需要看版式、图片或视觉效果时再渲染页面；
   - 图片：直接作为可查看材料；
   - docx / pptx：提取文本和图片；
   - 压缩包：解压后继续识别里面的文件。

8. 视觉类、视频类和混合作业会生成作品总览图；纯文字或文档类作业会为每个学生生成自己的文本评阅材料，并可合并成全班文字评阅包。

![虚构作品总览图](../assets/demo/fake-contact-sheet.png)

作品总览图可以帮助你先横向比较全班作业，发现明显优秀、明显敷衍、材料缺失或需要补看的学生。

对于纯文字或长文档作业，工具会先在每个学生的材料文件夹里生成 `review-text.md`，尽量合并该学生可提取的文本内容；然后可以生成 `assignment-review-text.md`，把未完成学生的文字材料按学生编号汇总到一个文件里。这个文件适合先整体阅读、横向比较和写草稿评阅，但不会跳过异常补看和用户确认。这个文本包有长度限制；如果内容不足、乱码或作业本身需要细读，仍应打开原始文档查看。

## 从草稿到正式结果

快速批阅不会直接写成最终成绩。工具会先生成草稿评阅，用来检查分数分布、评语质量和异常学生。

在用户确认之前，草稿不会转成正式评阅记录。对于边界分、明显高分、明显低分、材料缺失或代表图不充分的学生，建议继续补看关键材料。

![安全边界](../assets/demo/safety-boundaries.png)

## 最终输出

确认后，工具会把正式评阅记录保存在：

```text
result/
```

最终 Excel 默认导出到：

```text
outputs/
```

Excel 可以包含学生姓名、学号 / 编号、分数和评语。一个评阅记录文件里如果有多个作业，导出时会放到不同工作表里。

![Excel 输出预览](../assets/demo/excel-output-preview.png)


## 提示

- 工具不会在泛雅 / 超星网页上提交成绩。
- 工具不会替代老师的最终判断。
- 评分标准必须先确认。
- 学生名单必须来自真实压缩包或真实网页列表。
- 快速批阅先生成草稿，用户确认后才转成正式评阅记录。
- 真实学生数据、真实课程截图和真实作业作品不要提交到公开仓库或公开展示。
